# MiniDeliveryProject
이 프로젝트는 '미니 배달의 민족 플랫폼' 로 IT 웹 서비스 환경과 개발의 다변화에 따른 개발방식 스터디 및 직접 구현을 위하여 시작하였습니다. Spring Boot, JPA 등의 사용 경험 축적을 중점으로 두고 개발하였습니다. 이 프로젝트의 수행 경험으로 차기 사이드 프로젝트를 지속적 진행 예정
